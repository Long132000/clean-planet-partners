# Мобильная система работы с партнерами "Чистая Планета"

## Описание проекта
Мобильное приложение для автоматизации учета партнеров компании "Чистая Планета".

## Функциональность
- Управление партнерами
- Расчет материалов для услуг  
- История оказанных услуг
- Экспорт данных

## Технологии
- Kotlin
- Android SDK
- SQLite + Room
- MVVM архитектура
